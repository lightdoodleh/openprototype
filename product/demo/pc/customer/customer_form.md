# 客户表单 PRD（示例）

> 这是 openprototype 的示例 PRD。真实项目请按 `templates/docs/rules/form-page-prd-template.md` 编写。

## 一、页面信息

| 项 | 内容 |
|----|------|
| 页面名称 | 客户表单 |
| 页面文件 | customer_form.html |
| 页面类型 | 表单页（新增 / 编辑 / 查看） |

## 二、字段表

| 字段 | 控件 | 必填 | 只读(view) | 备注 |
|------|------|------|-----------|------|
| 客户名称 | 输入框 | 是 | 是 | |
| 联系人 | 输入框 | 是 | 是 | |
| 联系电话 | 输入框 | 否 | 是 | |
| 客户等级 | 下拉 | 否 | 是 | A / B / C 类 |
| 状态 | 下拉 | 否 | 是 | 启用 / 停用 |
| 备注 | 多行文本 | 否 | 是 | 占满一行 |

## 三、交互说明

- `?mode=view` 时所有输入控件与保存按钮**物理隐藏**（`display:none`），改显示只读文本（对应红线 4）。
- 保存写入 `demo-customer-list`（走 BaseDataManager，页面不直接读写 localStorage）。
