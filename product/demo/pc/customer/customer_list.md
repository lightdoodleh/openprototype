# 客户列表 PRD（示例）

> 这是 openprototype 的示例 PRD。真实项目请按 `templates/docs/rules/list-page-prd-template.md` 编写。
> 右侧 Agent 会自动携带「当前页面 + 当前 PRD」，可直接说"把某字段改成 XX"，Agent 会同步改页面与本 PRD。

## 一、页面信息

| 项 | 内容 |
|----|------|
| 页面名称 | 客户列表 |
| 页面文件 | customer_list.html |
| 页面类型 | 列表页 |

## 二、字段表

| 字段 | 说明 | 必填 | 备注 |
|------|------|------|------|
| 客户名称 | 支持模糊查询 | 否 | 查询条件 |
| 状态 | 启用 / 停用 | 否 | 下拉，来自状态常量 |
| 客户等级 | A / B / C 类 | 否 | 列表展示 |

## 三、操作按钮

| 按钮 | 说明 |
|------|------|
| 查询 | 按条件过滤列表 |
| 重置 | 清空查询条件 |
| 新增 | 跳转客户表单新增 |
| 编辑 | 跳转客户表单查看/编辑 |
